# Convert SASS/SCSS to LESS

> Convert SCSS (SASS) files to Less 

## Prerequisites:

	- NodeJS
	- Git

## Getting Started

### 1. Clone and install NPM package
```
git clone git@github.com:leduong/sass2less.git
cd sass2less
npm i
```

### 2. Copy your file to "sass" then convert
```
npm start
```

### 3. Check directory 'less'
